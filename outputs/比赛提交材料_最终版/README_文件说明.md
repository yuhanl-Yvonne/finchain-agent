# 比赛提交材料说明

本文件夹汇总了本次比赛建议优先使用的核心成果文件，按“结果、答辩、验图、模型附件”四类整理。

## 一、01_核心结果

### 1. `XGBoost融合白名单推荐表.xlsx`
- 用途：当前最重要的主结果表。
- 内容：融合了结构化特征与 GraphSAGE 图嵌入后的最终白名单推荐结果。
- 适用场景：比赛提交、汇报展示、老师查看最终推荐名单。

### 2. `XGBoost融合白名单推荐表.csv`
- 用途：上面主结果表的 CSV 版本。
- 内容：与 Excel 版一致，便于程序读取或快速筛选。
- 适用场景：需要再做筛选、排序、代码处理时使用。

### 3. `最终白名单人工核验表.xlsx`
- 用途：模型结果进入人工复核的工作底表。
- 内容：包含推荐企业、模型分数、图谱核验位、风险判断位、人工备注位。
- 适用场景：你自己或团队成员逐家核验后，形成最终正式名单。

### 4. `最终推荐简表.xlsx`
- 用途：给老师或评委快速浏览的简版名单。
- 内容：比完整推荐表更短，更适合快速展示。
- 适用场景：答辩现场、微信群发送、临时汇报。

## 二、02_答辩材料

### 5. `答辩汇报版结果说明.md`
- 用途：一页式结果说明文稿。
- 内容：概括图谱、GraphSAGE、XGBoost 融合模型与最终推荐结果。
- 适用场景：写 PPT 文字、汇报摘要、比赛答辩口径。

### 6. `答辩讲稿_融合模型版.md`
- 用途：直接可讲的答辩底稿。
- 内容：按“项目目标 - 图谱 - GraphSAGE - XGBoost - 结果 - 亮点”组织。
- 适用场景：答辩前练习、正式讲解。

### 7. `方法流程说明.md`
- 用途：方法论介绍材料。
- 内容：从数据整合、知识图谱、特征构建、图嵌入训练到融合排序的完整流程。
- 适用场景：PPT 方法页、老师问“你们怎么做的”时使用。

### 8. `最终入选名单说明.md`
- 用途：解释为什么这些企业会进入候选名单。
- 内容：名单结构特征、区域分布、产业链分布和人工核验建议。
- 适用场景：结果页说明、结论页扩展说明。

### 9. `SHAP处理方法说明.md`
- 用途：说明 SHAP 是如何接入最终融合模型的。
- 内容：包括 SHAP 计算逻辑、正负向解释生成方式以及如何写回结果表与图谱。
- 适用场景：答辩时回答“你们怎么做可解释性”的问题，或写入正式材料正文。

## 三、03_验图材料

### 10. `neo4j验图查询清单.md`
- 用途：在 Neo4j Browser 中直接验图的查询语句集合。
- 内容：按最终 XGBoost 融合分数查看企业、产业链、区域、风险以及 SHAP 解释的查询语句。
- 适用场景：现场演示图谱、截图、逐家核验。

## 四、04_模型附件

### 11. `xgboost_fusion_report.json`
- 用途：保存最终融合模型指标。
- 内容：训练样本数、验证集/测试集 F1、AUC、模型参数等。
- 适用场景：答辩时说明模型效果，或需要复核模型表现时使用。

### 12. `xgboost_fusion_feature_importance.csv`
- 用途：查看 XGBoost 主要依赖了哪些特征。
- 内容：特征重要性排序，能看出图嵌入和结构化特征的贡献。
- 适用场景：老师问“模型主要看什么特征”时使用。

### 13. `top20_xgboost_fusion_recommendations.csv`
- 用途：最终融合模型 Top20 候选清单。
- 内容：是主结果表的精简核心版，已包含 `SHAP正向解释` 和 `SHAP负向解释` 字段。
- 适用场景：快速抽取重点企业、制作 PPT 榜单页。

### 14. `graphsage_report.json`
- 用途：GraphSAGE 阶段的训练结果记录。
- 内容：图节点数、边数、验证集/测试集指标等。
- 适用场景：说明图神经网络阶段效果，体现项目技术深度。

### 15. `xgboost_fusion_shap_summary.csv`
- 用途：查看全局 SHAP 重要性。
- 内容：统计每个特征的平均绝对 SHAP 值和平均 SHAP 值。
- 适用场景：说明全局上哪些特征最影响模型判断。

### 16. `xgboost_fusion_shap_values.csv`
- 用途：保存全样本逐企业 SHAP 值。
- 内容：每家企业在各个输入特征上的 SHAP 贡献。
- 适用场景：深入分析单个企业的模型解释，或后续扩展可视化。

### 17. `xgboost_fusion_model.json`
- 用途：保存训练后的 XGBoost 模型。
- 内容：模型结构与参数，可用于复现预测和解释。
- 适用场景：代码复现、模型回载、继续扩展解释分析。

## 五、建议你优先看的顺序

1. 先看 `01_核心结果/XGBoost融合白名单推荐表.xlsx`
2. 再看 `02_答辩材料/答辩汇报版结果说明.md`
3. 然后看 `02_答辩材料/答辩讲稿_融合模型版.md`
4. 如果需要现场演示图谱，就打开 `03_验图材料/neo4j验图查询清单.md`

## 六、最核心的三个文件

- `XGBoost融合白名单推荐表.xlsx`：最终模型结果
- `最终白名单人工核验表.xlsx`：最终人工定稿表
- `答辩讲稿_融合模型版.md`：答辩直接可讲稿
